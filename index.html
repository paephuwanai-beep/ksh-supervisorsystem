คู่มือพยาบาลตรวจการ

Nurse Supervisor Guide — โรงพยาบาลชุมชน

Module 01

การบริหารอัตรากำลัง

  • ประเมินก่อนรับเวร: IPD ไม่เกิน 1:4 / ER ไม่เกิน 1:3
  • Patient Acuity: ประเมิน Level 1-3, on vent/drip
  • Float Staff: เกลี่ยคน → บันทึก Float Log
ห้ามหอผู้ป่วยวิกฤตมีพยาบาลคนเดียว
Module 02

ฉุกเฉินและอุบัติภัยหมู่

  • ประเมินขนาด: แยก 'ฉุกเฉินปกติ' กับ 'MCI'
  • ประกาศรหัส: Code Disaster → เปิด Command Post
  • START Triage: 🔴 ด่วนมาก 🟡 ปานกลาง 🟢 เล็กน้อย ⚫ ตาย
หลัก MCI: ช่วยผู้ที่มีโอกาสรอดก่อน
Module 03

ระบบการส่งต่อผู้ป่วย

  • เคส Refer: ผ่าตัดเฉพาะทาง, ต้องการ ICU, CT/MRI
  • รายงาน ISBAR: Identify / Situation / Background / Assessment / Recommend
เป้าหมาย: D-to-D เคสวิกฤตไม่เกิน 30 นาที
Module 04

คุณภาพ/ความปลอดภัย

  • จัดการทันที: รักษาความปลอดภัยผู้ป่วยก่อนเสมอ
  • Incident Report: บันทึก IR ในเวร ระบุความจริง ห้ามปิดบัง
'Just Culture': รายงานเพื่อเรียนรู้ ไม่ใช่เพื่อลงโทษ
Module 05

กฎหมายและสิทธิผู้ป่วย

  • รับมือร้องเรียน: พาไปห้องเงียบ → ฟังไม่ขัด → ไม่เถียง
  • นิติเวช: ตายผิดธรรมชาติ → แจ้งตำรวจ → ห้ามย้ายหลักฐาน
ไม่แน่ใจกฎหมาย ให้ปรึกษาผู้บริหารทันที

Checklist ก่อนส่งเวร

  • บันทึก Supervisor Log
  • เคลียร์เคส Refer
  • จัดอัตรากำลังเวรถัดไป
